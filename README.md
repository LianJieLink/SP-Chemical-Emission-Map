# SP-CE-MapV1

這是一個基於 Python 與 Folium 的地圖生成腳本，主要用於視覺化台灣各大科學園區（中科、竹科、南科等）周遭區塊，以及標示重要的環境、氣象監測站與周圍企業廠房的排放狀況。

## 🌟 功能重點

執行 `SP-CE-MapV1.py` 後，將產出一份互動式 HTML 靜態網頁地圖檔 (`index.html`)。地圖內建以下豐富圖層與標示功能：

### 🗺️ 1. 底圖切換
- **OpenStreetMap** (預設街道圖)
- **Google Satellite** (衛星影像)

*(可透過右上角控制面板自由切換)*

### 🏭 2. 廠區範圍 (紅色/淡藍色多邊形)
精準標示各大重點廠區與重要設施範圍：
- **中部科學園區 (中科)**：台積電 15A 廠、15B 廠、AP5 廠、台中零廢製造中心
- **新竹科學園區 (竹科) 及周遭**：台積電 F12 廠區 (P1&2, P4&5, P6, P7, P8)、F20 廠、AP3 廠 (龍潭)、AP6 廠 (竹南)
- **南部與高雄園區**：台積電 F18 廠區 (P1, P2, P7)、F22 廠
- **鄰近企業廠房 (淡藍色)**：包含華邦電子、台灣日東光學、均豪精密工業、永勝光學、橋椿金屬、和大工業、特典工具、矽品精密工業等廠區。系統自動從 `database` 資料夾讀取盤查清冊資料，並產生專屬可拖曳視窗彙整「煙道數量」以及所有「原料與排放化學物質」。若盤查資料缺失化學物質清單 (欄位為 N/A)，系統將在地圖上警示「未具備操作許可證、緊急應變計畫、毒化物/關注物質核可文件、環評報告書」。

### 🏭 2.5 列管煙道與化學物質盤查定位 (✅ 新增)
透過 Pandas 讀取排放資料庫 ：
- 直接於廠區範圍內以**藍色點位**精準標示所有單一煙口座標 (標示經緯度至小數 5 位、高度至小數 1 位)。
- 滑鼠懸停於藍點上，即可快速查閱對應該特定煙口所排放的受管制物質名稱！

### 📍 3. 測站位置分布與即時資訊面板
整合不同單位的環境測站，並以不同圖示嚴格區分：

- 🔵 **環境部測站 (藍色星號)**
  - 涵蓋：沙鹿、西屯、忠明、楠梓、左營、仁武、龍潭、新竹、頭份、朴子、嘉義、新港、善化、安南等站。
  - **✨ 極致即時的動態空品面板**：全面介接環境部最新版 API。資料獲取**已完全轉移至前端 JavaScript 自動處理**。只要您將 `index.html` 託管至 GitHub Pages 或任何靜態網頁空間，瀏覽器即會自動平行連線抓取各站點的 **PM10、PM2.5、NO2、SO2、O3、CO、溫度、濕度、風向、風速 (皆自動轉為台灣 LST 時間)**！
  - **🛡️ 嚴謹的最新資料過濾機制**：地圖前端會自動一一比對各項污染物的監測時間戳記 (`monitordate`)，過濾歷史紀錄，**保證資訊面板上絕對只顯示該測站最新一筆數據。**此外具備「**無資料欄位自動隱藏**」機制，遇到 `N/A` 的空值列將不會顯示，維持版面乾淨！

- 🔺 **特殊性工業區測站 (綠色三角形)**
  - 涵蓋：中科周遭 (陽明國小、國安國小、都會公園、中科實中)、竹科周遭 (新安、靜心湖、力行、興業、竹南東/西/南/北)、南科周遭 (公13、公19、公29、南科實中)、嘉義東石、油廠國小、半屏山加壓站、園區東南站、園區北站... 等數十個重點觀測站。
  - **✨ 工業區監測數據同步連線**：這批特殊測站現在也已導入與環境部測站相同的 JavaScript 前端平行連線抓取架構 (`aqx_p_268` API)，能即時在網頁上載入特工區的第一手污染清單。

- ☁️ **氣象署測站 (橘色雲朵)**
  - 包含：大雅(自動)、高雄(人工)、龍潭(自動)、新竹市東區(自動)、寶山(自動) 等十多個人工與自動測站。

### ✨ 4. 創新的資料追蹤面板 (Popup UI)
- **多重視窗交叉比對**：上述即時資料面板皆設計為「點擊開啟 (Click-to-open)」，並**開放允許無限數量同時展開於地圖上**，不再有傳統地圖點開 A 就關閉 B 視窗的限制！
- **自由拖拉面板 (Drag & Drop)**：每個測站數據面板的標題列皆加入了原生拖曳功能。您可以點擊按住面板標題列，將面板自由任意拖動放置在地圖畫面的任何角落，輕鬆客製專屬的數據儀表板佈局組合。

### 🛠️ 5. 地圖互動元件
- **圖層控制開關 (Layer Control)**：自由啟閉廠區或各類測站圖層。
- **全螢幕顯示 (Fullscreen)**：支援一鍵進入全螢幕模式。
- **游標座標 (Mouse Position)**：即時顯示滑鼠指向的精確經緯度。
- **量測工具 (Measure Control)**：支援地圖上的距離 (公尺/公里) 與面積 (平方公尺) 測量。

## 📦 系統需求與執行方式

### 1. 安裝依賴套件
請確保你的 Python 環境已安裝 `folium`、`pandas` 以及 `openpyxl` (讀取 Excel 所需)：
```bash
pip install folium pandas openpyxl
```

### 2. 生成地圖
在終端機中執行 Python 腳本：
```bash
python SP-CE-MapV1.py
```
> **注意**：執行此腳本僅負責**建立介面與繪製基礎靜態圖層**。環境部 API 的連線與即時抓取是交由產生的 `index.html` 在使用者瀏覽器端動態載入的，大幅減輕了伺服器端（與 Python 端）的負擔。
